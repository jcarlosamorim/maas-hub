As 4 Causas de Metodologia

Para entender algo em primeiro princípio, responda 4 perguntas fundamentais.

Do que é feito?
Causa Material
DECISÕES SEQUENCIADAS
Cada passo é uma decisão que o expert tomou. A sequência dessas decisões é o material bruto.
Qual a estrutura?
Causa Formal
INPUT → TRANSFORMAÇÃO → OUTPUT
Repetido em cascata. Cada output vira input do próximo. Cadeia de transformações.
O que faz funcionar?
Causa Eficiente
O AGENTE QUE EXECUTA
Pode ser: Humano, IA, Sistema, UI/UX, ou combinação híbrida.
Para que serve?
Causa Final
REDUZIR FRICÇÃO A → B
Metodologia existe para que alguém chegue a B sem redescobrir as decisões do expert.
